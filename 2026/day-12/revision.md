# Breather & Revision

## Processes & services

* ps - Lists all running processes on the system.
* systemctl status - Displays the status of a specific service (whether it is Active, Failed or Inactive).
* journalctl -u <service> - Displays logs for a specific service,it is helpful for debugging issues.

## File skills

* echo - It allows users to display lines of text or strings that are passed as argument
* chmod - The "chmod" command modifies the read, write, and execute permissions of specified files and the search permissions of specified directories
* chown - The "chown" command in Linux is used to change the owner and/or group associated with a file or directory. 
* ls -l - It is used to list information about files and directories within the file system.
* cp - The cp (copy) command in Linux is used to duplicate files or directories from one location to another within the file system.
* mkdir -  The mkdir command in Linux stands for “make directory” and is used to create new folders quickly and efficiently from the terminal.
